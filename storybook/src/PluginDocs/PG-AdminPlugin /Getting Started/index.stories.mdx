# Getting Started

##

 The **PG-Admin** plugin sets up a **PG-Admin dashboard** that you can access via your browser. This dashboard provides an easy-to-use interface for managing your **databases**. With this plugin, you can perform various database management tasks, such as creating tables, running queries, and managing user permissions. The dashboard is accessible from anywhere, making it convenient for you to manage your databases from anywhere with an internet connection.
